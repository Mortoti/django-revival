# Django Revival

> I have built things. Now I am going back to the roots to master the craft properly.

## What this is

I am not starting from zero. I have shipped Django projects, written APIs, and solved real problems. But there is a difference between making things work and truly understanding them. This repository is the deliberate effort to close that gap, going deep on Django architecture, scalability, and system design until the fundamentals are not just familiar but second nature.

## Structure

```
.
├── forge/          # Code-along with courses, following the material closely
└── lab/            # Practice projects, applying concepts independently
```

**`forge/`** is where I work through course material. Code here mirrors the tutorial closely and is organized by course and module.

**`lab/`** is where I practise. Projects here start simple and grow in complexity as my skills develop. This is where the real learning happens.
